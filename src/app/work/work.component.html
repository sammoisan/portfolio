Devine la toune

Devine la toune is inspired by the board game Devine the sogne where players draw cards and read the lyrics to the others players. The lyrics are translated from English to French, so players have to translate the lyrics back to the original language to find the song. The idea behind Devine la toune is to keep up to date with new songs. It use Musixmatch API to get the most popular song of the moment and Google Cloud Translation API to translate the lyrics. I'm currently working with the Spotify API to add a new feature where user can import their own playlist.
